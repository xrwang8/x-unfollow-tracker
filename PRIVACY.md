# 隐私政策

**最后更新：2026-07-14**

## 概述

X 取关追踪器（以下简称"本扩展"）尊重并保护用户隐私。本隐私政策说明本扩展如何处理用户数据。

## 数据收集

**本扩展不收集任何个人数据。**

具体来说，本扩展：
- ❌ 不收集个人身份信息（姓名、邮箱、电话等）
- ❌ 不收集浏览历史
- ❌ 不收集位置信息
- ❌ 不收集支付信息
- ❌ 不收集账号密码或认证凭证

## 数据存储

本扩展在您的浏览器本地存储以下数据：

1. **关注者快照**：您在 X 上的关注者列表（用户名、昵称、头像 URL）
2. **关注快照**：您在 X 上关注的用户列表
3. **扩展设置**：您的偏好设置（如对比天数）

**存储位置**：所有数据存储在 `chrome.storage.local`，仅存在于您的设备上。

**数据保留**：快照自动保留 30 天，超期自动删除。卸载扩展会清除所有数据。

## 数据传输

本扩展的网络请求：

1. **向 x.com 发送请求**
   - 目的：调用 X 的 followers/list 和 friends/list 接口获取关注者列表
   - 使用的凭证：您浏览器中已存在的 x.com 会话 cookie
   - 接收方：x.com（Twitter/X 官方服务器）

2. **不向任何第三方服务器发送数据**
   - 本扩展没有自己的后端服务器
   - 不向任何第三方分析服务发送数据
   - 不向任何第三方广告服务发送数据

## 权限说明

本扩展申请以下权限：

| 权限 | 用途 |
|------|------|
| `storage` | 在本地存储关注者快照，用于历史对比 |
| `cookies` | 读取 x.com 的会话 cookie（ct0 字段），用于调用 X 接口 |
| `host_permissions (x.com)` | 访问 X 的接口以获取关注者列表 |
| `scripting` | API 失效时注入备用抓取脚本（仅在用户点击检查按钮时） |

## 数据安全

- 所有数据仅存储在您的设备上
- 不经过任何中间服务器
- 不会被上传或分享给任何第三方
- 数据随扩展卸载而删除

## 第三方服务

本扩展不集成任何第三方服务，包括但不限于：
- 分析工具（Google Analytics 等）
- 广告网络
- 社交媒体插件
- 支付处理器

## 儿童隐私

本扩展不针对 13 岁以下儿童，也不故意收集儿童的个人信息。

## 政策变更

如果隐私政策发生变更，我们会：
1. 更新本文档的"最后更新"日期
2. 在 GitHub 项目主页发布公告
3. 重大变更会在扩展更新说明中注明
