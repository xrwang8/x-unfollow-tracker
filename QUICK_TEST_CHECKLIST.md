## 测试步骤

### 第 1 步：验证 API 可用性（1 分钟）

**当前状态**：测试页面已在浏览器打开

**操作**：
1. 在新标签页打开 https://x.com 并登录
2. 回到测试页面（`/tmp/test-x-unfollow-tracker.html`）
3. 点击"检查登录状态"按钮
4. 点击"测试 followers/list API"按钮

**预期结果**：
- ✅ 绿色提示：API 正常 → 扩展会快速工作（3-5 秒）
- ⚠️ 黄色提示：API 失效 → 扩展会自动降级（慢但能用）

---

### 第 2 步：加载扩展到 Chrome（2 分钟）

**操作**：
1. 打开新标签页，地址栏输入：`chrome://extensions/`
2. 右上角开启"开发者模式"
3. 点击"加载已解压的扩展程序"
4. 选择目录：
   ```
   /Users/xrwang/go/src/github.com/xrwang8/x-unfollow-tracker/.output/chrome-mv3
   ```
5. 确认扩展加载成功

**检查点**：
- [ ] 扩展列表显示"X 取关追踪器"
- [ ] 版本号：0.1.0
- [ ] 工具栏出现蓝色图标
- [ ] 没有错误提示

---

### 第 3 步：测试基础功能（2 分钟）

**前置条件**：已登录 x.com

**操作**：
1. 点击工具栏的扩展图标
2. popup 窗口打开
3. 点击"检查关注者"按钮
4. 观察进度提示

**预期结果**：
- [ ] 显示"正在拉取关注者列表..."
- [ ] 进度显示 `[API] 已拉取 XXX 人...` 或 `[页面滚动] 已拉取 XXX 人...`
- [ ] 完成后显示"快照已保存：XXX 人"
- [ ] 快照列表出现新记录

**耗时**：
- API 方式：3-5 秒

---

### 第 4 步：测试 CSV 导出（1 分钟）

**前置条件**：至少有 2 个快照

**操作**：
1. 选择分析模式"谁取关了你"
2. 点击"查看结果"
3. 如果有结果，点击右上角"导出 CSV"按钮
4. 用 Excel 打开下载的文件

**检查点**：
- [ ] CSV 文件成功下载
- [ ] 中文显示正常（UTF-8 BOM）
- [ ] 包含：Username, Name, Profile URL
