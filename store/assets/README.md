# Chrome Web Store 素材制作指南

本目录用于存放 Chrome Web Store 上架所需的图片素材。

## 必需素材

### 1. 扩展图标 ✅ 已完成
- **尺寸**：128×128 px
- **位置**：`public/icon/128.png`
- **状态**：✅ 已有

### 2. 截图（Screenshots）
- **尺寸**：1280×800 px 或 640×400 px（推荐 1280×800）
- **格式**：PNG 或 JPG
- **数量**：至少 1 张，推荐 3-5 张
- **状态**：⬜ 待截图

**建议截图内容**：
1. **主界面** - 显示"检查关注者"和"检查你的关注"按钮
2. **拉取进度** - 显示"[API] 已拉取 200 人..."的进度提示
3. **快照列表** - 显示已保存的关注者快照和关注快照
4. **对比结果** - 显示"这些人取关了你"的名单界面
5. **CSV 导出** - 显示导出按钮和导出后的 Excel 文件

**截图步骤**：
1. 加载扩展到 Chrome（开发者模式）
2. 点击扩展图标打开 popup
3. 使用 macOS 截图工具（Cmd+Shift+4）或 Chrome DevTools 截图
4. 裁剪为 1280×800（保持 popup 居中，周围留白）
5. 保存到 `store/assets/screenshots/`

### 3. 小图块（Small promo tile）
- **尺寸**：440×280 px
- **格式**：PNG 或 JPG
- **状态**：⬜ 待制作

**设计要求**：
- 背景：渐变蓝色（符合 X/Twitter 品牌色）
- 图标：扩展 128×128 图标居中或左侧
- 文字：
  - 主标题："X 取关追踪器"（粗体，白色）
  - 副标题："3 秒检测 500 人 | 快 150 倍"（白色）
- 可选装饰：向上/向下箭头（表示关注变化）

**制作工具**：
- Figma / Canva（在线）
- Photoshop / Sketch（本地）
- 或使用 AI 生成（MidJourney / DALL-E）

### 4. 大图块（Marquee，可选）
- **尺寸**：1400×560 px
- **格式**：PNG 或 JPG
- **状态**：⬜ 可选（不是必需）

**说明**：大图块会显示在 Chrome Web Store 的精选位置，如果不上传也不影响上架。

## 快速截图模板（用于临时测试）

如果暂时没有真实截图，可以用以下方式快速生成占位图：

```bash
# 使用 ImageMagick 生成纯色占位图
convert -size 1280x800 xc:#1DA1F2 -gravity center \
  -pointsize 60 -fill white -annotate +0+0 "Screenshot 1" \
  screenshot-1.png
```

或在线生成：https://placeholder.com/

## 检查清单

上架前确认：

- [ ] 至少 1 张截图（推荐 3-5 张）
- [ ] 截图尺寸正确（1280×800 或 640×400）
- [ ] 小图块 440×280
- [ ] 所有图片清晰无模糊
- [ ] 截图内容真实（不要用 PS 伪造功能）
- [ ] 图片文件名规范（小写字母 + 连字符）

---

## 提示

1. **真实性**：Chrome 审核会检查截图与实际功能是否一致，不要夸大或伪造
2. **清晰度**：截图保存为 PNG 格式，避免 JPG 压缩导致模糊
3. **隐私**：截图里不要出现真实用户的隐私信息（可以用测试账号或打码）
4. **统一性**：所有截图使用相同的浏览器主题和 popup 尺寸
